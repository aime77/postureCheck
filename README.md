# Posture Check
## Overview
A sophisticated posture game which tracks body points while the user performs YouTube video stretches.

## Technologies
* Semantic-ui
* Tensorflow.js
* React.js
* Mongoose
* Node
* Express
* Axios
* Passport
* GoogleAuth

## Implementation
